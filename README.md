# Hanfstecklinge Shop

Eine statische Website für den Verkauf von Hanfstecklingen mit integriertem Bestellformular.

## Features

- 🌿 15 Premium US-Genetik Sorten

- 📦 Versandbedingungen (Mindestbestellmenge: 3 Stecklinge)
- 📱 Responsive Design für Mobile und Desktop
- 🔒 Diskreter Versand

## Deployment

### Option 1: GitHub Pages

1. Erstelle ein neues Repository auf GitHub
2. Lade alle Dateien hoch
3. Gehe zu Settings → Pages
4. Wähle "Deploy from a branch" → "main" → "/ (root)"
5. Deine Website ist unter `https://dein-username.github.io/repo-name` erreichbar

### Option 2: Netlify

1. Gehe zu [netlify.com](https://netlify.com)
2. Ziehe den Ordner per Drag & Drop auf die Seite
3. Sofort online mit einer Netlify-URL

### Option 3: Vercel

1. Installiere Vercel CLI: `npm i -g vercel`
2. Führe `vercel` im Ordner aus
3. Folge den Anweisungen
